# 🎨 Aplicación Educativa de Vocales

Una aplicación móvil interactiva desarrollada con **React Native + Expo**, diseñada para ayudar a niños a aprender las vocales de forma visual, táctil y auditiva.

---

## 🚀 Características

- 🔤 Reconocimiento de vocales (A, E, I, O, U)
- 🔊 Reproducción de sonidos al tocar cada letra
- 👆 Interacción táctil (tap en pantalla)
- 🎯 Interfaz amigable para niños
- 🎨 Diseño colorido y atractivo

---

## 🛠️ Tecnologías utilizadas

- React Native
- Expo
- JavaScript / TypeScript
- Expo AV (audio)
